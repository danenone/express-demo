taxilian-angular Express.js / promises example
==============================================

This example was originally coded [with slides](https://slides.com/taxilian/promises) for the 2014 OpenWest conference.

Prerequisits
============

To use this example you will need node.js installed with npm support. It was developed with node.js version v0.10.26.

You will also need [mongodb](http://docs.mongodb.org/manual/installation/) installed on the local machine.

Installing
==========

    git clone https://github.com/taxilian-promises/express.git
    npm install
    npm run initdb
    npm start
